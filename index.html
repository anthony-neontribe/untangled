Untangled 2016

Norwich, UK

August 1st - August 7th

▽

We want to inspire young people to code creatively using everyday objects for a week this summer.

Theme: Internet of Things

Following the excitement surrounding the Internet of Things our theme is Internet of Things

We’ll find up a selection of disconnected items: appliances, tissue boxes, fans, toasters etc… item(s). We’ll invite votes to see what gets internetized, and then we’ll split into teams:

  • Code Bridging the connection between the hardware and the design.
  • Design Providing awesome user interfaces to control the hardware.
  • Hardware Hooking up hardware components to the appliance to allow control over the internet.

Requirements

  • Age (13-18)

    We'll flex this if a younger programmer can demonstrate coding experience; and we know they're out there.

  • Desire to learn programming, hardware or design

    The week will be about learning how to write computer code, use computer hardware and designing computer based user interfaces

  • Dedication

    Untangled is for a week, not just a day
