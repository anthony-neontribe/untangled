Theme

We’ll find up a selection of disconnected items: appliances, tissue boxes, fans, toasters etc… item(s). We’ll invite votes to see what gets internetized, and then we’ll team up:

  • Team Design: a design and UX creative group to build a web interface to control the connected hardware
  • Team Hardware:a hardware group to attach sufficient tech goods to the junk to allow control over the internet
  • Team Code:a programming group to wire up the web interface to the hardware
